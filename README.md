# Tender Smart Search

Локальный MVP для умного поиска товаров и услуг на `goszakup.gov.kz`.

## Что улучшено

- Можно писать человеческий запрос: `сервер для больницы`, `лицензия 1С`, `ноутбук i7`
- Запрос разбирается на ключевые термины
- Система расширяет запрос синонимами
- Поиск идет по нескольким фразам
- Лоты дедуплицируются
- Результаты сортируются по релевантности
- Учитывается минимальная сумма

## Запуск

```bash
python3 app/main.py
```

Открой:

```text
http://127.0.0.1:8090
```

## Запуск на сервере

На облачном сервере приложение должно слушать внешний интерфейс:

```bash
HOST=0.0.0.0 PORT=8090 python3 app/main.py
```

Инструкция для Hetzner: [deploy/hetzner.md](deploy/hetzner.md).
